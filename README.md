# Tala Mobile Lending Uganda

Database project simulating a microfinance portfolio in Uganda, including:
- Customers
- Loan officers
- Loans issued
- Repayments

## Purpose
Designed for BI, analytics, and executive decision-making. Sample dataset with anonymized personal information.

## Files
- `schema_and_data.sql` – Database schema + sample data
- `analytics_queries.sql` – KPI and portfolio queries
- `README.md` – Project overview
